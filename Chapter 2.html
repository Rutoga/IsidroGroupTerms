Chapter 2
Networking Fundamentals

Repeater

Regenerates signals

Hub

Concentrate connections and regenerate a signal

Bridge

Convert network transmission data formats as well as perform basic data transmission management

Switch

Add more intelligence to data transfer management

Router

Routing and other services

Physical topology

actual layout of the wire

Logical topology

Defines how the media is accessed by the hosts for sending data

Protocol

Formal description of a set of rules and conventions that govern a partifuclar aspect of how devices on a network communicate.

LAN

Allows many users to access high-bandwidth media

WAN

Provide full-time remote resources connected to local services

MAN

network that spans a metropolitan area such as a city.

SAN

is a dedicated, high performance network used to move data between servers and storage resources.

VPN

private network that is constructed within a public network.

Bandwidth

is the measure of how much information, or bits, can flow from one place to another in a given amount of time, or seconds..

Throughput

Refers to actual measured bandwidth.
